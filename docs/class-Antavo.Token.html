Class Token

Packs/unpacks a signed token with custom data.

Namespace: Antavo
Example:


$token = new Antavo\Token($token_secret);
$token->setData(array('user_id' => 1));
echo $token->getToken();


Example:


$token = new Antavo\Token;
$token->setToken($input_token);
var_dump($token->getData());

# setExpirationTime( integer $expires_at )

Sets expiration time for the token.

Sets expiration time for the token.

Parameters

$expires_at

Unix Timestamp of expiration time. If given value is smaller than or equals to 30 days (in seconds) then it is handled as a relative time-to-live value. 0 means it will never expire. Float values are casted to integer, while non-numeric values are discarded.
